Bika LIMS
---------

### The meaning of Gaob

In the tradition of Chief Inkosi Bika LIMS 2 using a title for a release, this edition is code-named Gaob Bika LIMS 3. Gaob, King, is taken from Khoi San Nama, language of the world's first people.
