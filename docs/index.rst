============
senaite.core
============

**Documentation for developers**

`senaite.core`_ is an add-on for `Plone CMS`_ that provides the main logic,
contents, behavior and user interface for `SENAITE LIMS`_.

It is intended to be imported automatically as a dependency of
`senaite.lims Meta Installation Package`_ and/or other SENAITE products, and it
should **not** be installed alone.
